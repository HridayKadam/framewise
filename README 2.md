# FrameWise
Real-time, on-device AI composition assistant built for the WWDC 2026 Swift Student Challenge.

- 100% on-device, no networking
- Uses AVFoundation, Vision, CoreML, ARKit, RealityKit, Combine, CoreMotion, CoreHaptics, PhotoKit, CoreImage, AVSpeechSynthesizer, SwiftUI

Build with Xcode 26.2+ on iOS 18+.
